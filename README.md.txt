# Système de Transfert d'Argent - Frontend

## Description
Ce dépôt contient l'interface graphique du système de gestion de transfert d'argent.

## Technologies utilisées
- Python 3
- CustomTkinter
- Tkinter

## Architecture
Interface graphique séparée du backend (architecture en couches).

## Installation

1. Cloner le projet
2. Installer les dépendances :

pip install -r requirements.txt

3. Lancer l'application :

python login.py
